Projects

Project 1 : Word Divider in Assembly language

Creating a word divider in assembly language involves reading a string, identifying word boundaries (such as spaces or punctuation), and storing each word separately. The process includes initializing pointers and buffers, looping through the characters, and handling end-of-string conditions. This can be implemented in x86 assembly with the NASM syntax to manipulate memory directly for processing and storing words.

Project 2 : Online Job Portal

A job portal using SQL and PHP allows users to search and apply for jobs online, while employers can post job listings. SQL is used to manage the database, storing user details, job postings, and applications. PHP handles the server-side logic, enabling dynamic interactions between users and the database, such as job searches, applications, and account management.
